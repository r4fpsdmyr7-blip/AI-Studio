# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅        |

## Reporting a Vulnerability
Please report security vulnerabilities to [security email] instead of opening a public issue.
We will acknowledge receipt within 48 hours and provide a fix within 30 days.

## Security Best Practices
- Always verify script integrity before execution
- Review `env-install.sh` before running dependency installation
- Use `--keep-data` flag when testing uninstall operations
